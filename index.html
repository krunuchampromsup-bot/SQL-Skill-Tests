SQL VIEW Master Class

ศูนย์การเรียนรู้ระบบฐานข้อมูลระดับมืออาชีพ โดย ครูนุช Smart Com

รูปแบบคำสั่งพื้นฐาน
CREATE VIEW view_name AS
SELECT column1, column2 FROM table_name WHERE condition;

คำอธิบาย: ใช้สำหรับสร้างตารางเสมือน (Virtual Table) เพื่อกรองข้อมูลที่ต้องการจากตารางหลัก

ตัวอย่างคำสั่งผลลัพธ์
SELECT * FROM it_staff; แสดงรายชื่อเฉพาะพนักงานฝ่าย IT เท่านั้น
การเพิ่มข้อมูลผ่าน VIEW
INSERT INTO view_name (column1, column2)
VALUES (value1, value2);

คำอธิบาย: ข้อมูลจะถูกส่งผ่าน VIEW ไปจัดเก็บยังตารางหลัก (Base Table) โดยอัตโนมัติ

การแก้ไขข้อมูลผ่าน VIEW
UPDATE view_name
SET column1 = value1 WHERE condition;

คำอธิบาย: เมื่อแก้ไขข้อมูลผ่าน VIEW ค่าในตารางหลักจะเปลี่ยนแปลงตามไปด้วย

การลบข้อมูลผ่าน VIEW
DELETE FROM view_name WHERE condition;

คำอธิบาย: ลบแถวข้อมูลออกจากตารางหลักผ่านเงื่อนไขที่กำหนดใน VIEW
